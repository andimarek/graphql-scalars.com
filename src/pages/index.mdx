# Welcome to GraphQL-Scalars.com

[GraphQL supports a specification URL for custom Scalars](https://github.com/graphql/graphql-spec/pull/649) 

This page hosts community definied custom Scalars which can referenced via

`https://www.graphql-scalars.com/<SPECIFICATION-NAME>` 


For questions and general feedback please open a GitHub Issue at:
